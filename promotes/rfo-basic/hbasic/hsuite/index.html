hSuite v3.30

hSuite is a set of utilities for compiling hBasic or a custom version to an APK or AAB for Android.
License

You can compile a Basic program into
an APK for installing on an Android device
or straight to AAB for Google Playstore upload.
hSuite can do this without you having to use any intermediary tools.

hBasic is a Basic Interpreter forked from Rfo-Basic 01.91 (De Re Basic) for Android originally by Paul Laughton.
