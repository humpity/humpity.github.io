hBasic Manual
Colors Strings for various commands.

These color strings may be used in place of the legacy decimal values in GR.OPEN and GR.COLOR
A <color> string may either be

hex #AARRGGBB as alpha, red, green, blue    e.g #FF0000FF
or
hex #RRGGBB as red, green, blue (alpha assumed to be solid)     e.g #0000FF
or
any one of the 564 HTML color names listed below or original website here.    e.g "blue"

These color strings are searched for in the above order.

Alpha channels for color Names

A color name can be post-fixed with an alpha-channel value e.g

CONSOLE.SET "TEXTCOLOR darkcoffee.169"
is the same as
CONSOLE.SET "TEXTCOLOR darkcoffee#A9"
is the same as
CONSOLE.SET "TEXTCOLOR #A93B2F2F"

#hh    A 'H'ash ( # ) begins the alpha value as HEX.
.ddd   A 'dot'     ( '.' ) begins the alpha value as Decimal.

These colors are listed on https://www.computerhope.com/htmcolor.htm#color-codes

There are 564 color names covering 554 unique colors.
