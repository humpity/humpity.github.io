Moving to Manjaro Linux - June 2015.

I am starting to migrate from Lubuntu to Manjaro which is based on Arch Linux.
The migration may take a long time since I am already heavily invested into Lubuntu.

The reason being is that the systemd controversy is still going on since 2012.

"systemd" is a startup system designed to replace 'init' and an alarming number of other utilities into one blob.

The fear is that the systemd "..design is contrary to the Unix philosophy of interconnected utilities" ."[66] and that "other components ..are forced to be used due to a network of interlocking dependencies" [7] and may eventually lead to bloat.

After trying out a boatload of non-systemd distros, I have finally found one that is easy to install and compatible with most of things I use with Lubuntu. Manjaro is also one of the rare distros that let's you boot with ExtLinux instead of Grub.

Aside from my own installation experience below, there is also a DistroWatch article here.

Installing Manjaro

Manjaro's default distro is systemd, but the community maintains an OpenRC version.
The OpenRC+XFCE+OpenBox ISO CD image can be downloaded from SourceForge HERE.

Steps.
1. Download the ISO
2. Write it to USB thumbdrive using UNetbootin.
3. Boot the thumbdrive into the live CD. (select 'start' instead of 'default')
4. Insert your target partition (can be another USB drive or a hardrive) and make sure it's unmounted.
5. run 'gparted' to find your target partition and format it as EXT4. Also add the 'boot' flag.
    note the name of your target partition e.g sdb.

6. Click the Install icon on the desktop. It will open a terminal to run the text based installer.
7. Set date/time.
8. Disk Preparation - Jump straight to: Set Mountpoints and choose your target partition (e.g sdb).
    'create filesystem'? = NO.
    'additional filesystem' = NO.
    e.g /dev/sdb1: ext4:no:<label>:NONE...  OK.
    use FSUUID as device name scheme.
9. Install System.
10. Configure:
    should <user> get suso rights? = YES.
    Locale - SKIP
    Keyboard map - SKIP.
    Config system - SKIP.
    <..configuring locales..>
    <..building initramfs..>
11. Install Bootloader.
    e.g
    BIOS
       SYSLINUX_BIOS SYSLINUX/EXTLINUX
    Select boot device
       /dev/sdb/         (this will install into the MBR)
       /dev/sdb1/        (this will install into the VBR of the partition - if you already have an MBR Manager)

Generally it is safer to install into the VBR, since most PCs will boot the VBR anyway if there is no manager in the MBR and the VBR is marked with the 'boot' flag.

    'extlinux editor' - CTRL-X to exit.

12. Quit.

** That's it ! **.
