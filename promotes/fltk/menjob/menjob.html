MenJob
A batch Job Gui front-end and Script Generator for MEncoder

Makes for easy encoding.

MenJob is written in c++ and FLTK.


 Project Stage : Idea Infancy Basic. v3.

Features
Gui to set commonly used parameters for MEncoder

Batch job for multiple files (in directory).

Xvid encoding
LAVC (mpeg4/divx) encoding
x264 encoding (since v2)
2 pass encoding (since v3)

single executable.  settings saved to $home,  won't mess up your OS.

Instructions
Input by gui. 'Generate' scripts. Run 'dobatch' in a terminal.
